Array in Javascript

Array Methods

  1. Push : Add element at the end of an array
  2. Pop : Remove element at the end of an array
  3. Shift : Remove first element of an array
  4. Unshift : Add element at the start of an array
  5. Slice : use to copy the part of an array or complete array and make new array
  6. Splice : use to add, remove element from anywhere in an array.
