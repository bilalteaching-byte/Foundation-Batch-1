Data Types

  1. String : Jo Inverted Commas mein ho e.g let name="Test"
  2. Number : any number not in string e.g let age = 39
  3. Boolean : true or false e.g let isActive = false
  4. Object : multiple values under curly/square brackets e.g let info = {}
  5. undefined : variable is declared but have no value e.g let a;
  6. null : give null value to any variable
